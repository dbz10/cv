\talkentry
{Strange Metals}
{}
{
\vspace{-10pt}{
\begin{itemize}
\item Designed a model to reproduce and explain certain phenomena observed in high temperature superconducting materials.
\item Wrote an implementation of the Density Matrix Renormalization Group algorithm for simulating one dimensional quantum systems to study the model numerically.
\item Performed massively parallelized large scale numerical analysis using HTCondor. 
\end{itemize}

\talkentry

{}
{
\vspace{-10pt}{
\begin{itemize}

\item Designed and implemented a combination of Matrix Product State and Monte Carlo algorithms in C++ to measure entanglement entropy.
